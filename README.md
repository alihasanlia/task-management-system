# Task Management System

This project is an implementation of a task management system using advanced data structures.

## Features
- Insert, delete, and update tasks
- Query task by ID
- Sum of task values in a given ID range
- Detect task time conflicts
- Print internal trees structure

## Data Structures Used
- B-Tree (task storage by ID)
- Segment Tree (range queries)
- Interval Tree (conflict detection)

## Commands
- InsertTask id startTime endTime value
- DeleteTask id
- UpdateTask id startTime endTime value
- QueryTaskId id
- QueryTaskSum id1 id2
- PrintTrees

## Laguage
- Java
